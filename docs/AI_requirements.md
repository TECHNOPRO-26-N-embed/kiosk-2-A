## 2. 要求仕様書（顧客ヒアリング結果）

### 2.1 顧客の要望（ヒアリングメモ）

> ここに、顧客（講師）へのヒアリングで得た要望を箇条書きで記録してください。

* 商品をカテゴリで選べる　野菜、果物、などから選択する形
* 商品種類が最大10種類
* 税率別(8%,10%)
* 支払方法　複数あればありがたい実装できなくても拡張できれば
* CSVにログを保存する
### 2.2 機能要件

| 機能ID | 機能名 | 概要 |
| :---: | :--- | :--- |
| F01 | カテゴリ選択機能 | 野菜・果物などのカテゴリから商品を選択できるようにする。 |
| F02 | 商品管理機能（最大10種類） | 取り扱い商品を最大10種類まで登録・表示・選択できるようにする。 |
| F03 | 税率別計算機能 | 商品ごとに8%または10%の税率を設定し、会計時に税額・合計金額を計算する。 |
| F04 | 会計ログCSV保存機能 | 取引情報（日時、商品、数量、金額、税率）をCSVへ保存する。 |
| F05 | レシート表示機能 | 購入明細、税率別内訳、合計金額を画面に表示する。 |
| F06 | お客様呼び出し機能 | エラーやトラブル時に店員を呼び出す機能を提供する。 |
