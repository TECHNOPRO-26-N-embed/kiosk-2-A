# AIテスト仕様書（ai_test_spec）

## テストケース一覧


| テスト項目             | 入力値例            | 期待結果                                      |
|----------------------|----------------------|---------------------------------------------|
| 商品カテゴリ選択         | 1 → 3 → 3        | 「おにぎり x 5」がカートに追加され、在庫が減少する |
| 商品カテゴリ異常入力      | 9                | 「Invalid choice. Please try again.」と表示    |
| 商品検索（部分一致）      | コー              | 「コーラ」が候補に表示される                    |
| 商品検索（該当なし）      | XYZ               | 「該当する商品がありません。」と表示される       |
| 支払金額不足            | 1000               | 「入金額が不足しています。」と表示される         |
| 支払金額正常             | 合計金額以上の金額  | 正しくおつりが表示される                       |
| 数字以外の入力（メニュー）| abc                | 「数値を入力してください。」と表示される        |
| 数量0入力               | 0                |「数量は1以上の整数を入力してください。」と表示される|
| レシート発行選択         | 1                 | レシート内容が表示される                        |
| レシート発行未選択       | 2                 | 「ありがとうございました！」のみ表示される        |
